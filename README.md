# Centipede

## Summary
	A clone of Centipede, based in space. Players attempt to destroy waves of centipedes
	which grow increasingly aggressive as the game progresses.

## About
	Built in HTML5/Canvas. Also uses jQuery and Underscore.js.

	All other code and art assets are original.

## Features
	* Collisions are handled by compact [hitbox](https://github.com/mgukowsky/centipede/blob/gh-pages/lib/hitbox.js) objects, which are garbage collected manually.
